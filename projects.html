01 — Full-Stack ML · Feb 2026

Stock Prediction System

End-to-end ML application predicting next-day stock volatility using Random Forest models trained on 15 technical indicators (RSI, MACD, Bollinger Bands, ATR), optimized via RandomizedSearchCV with TimeSeriesSplit cross-validation. NLP sentiment pipeline uses FinBERT with exponential time-decay weighting over 72-hour news windows to aggregate market signals. Dockerized FastAPI backend and Next.js/TypeScript frontend with multi-stage builds; integrated NVIDIA Nemotron LLM via OpenRouter API for natural language investment recommendations. Production features include automatic model retraining (7-day intervals), rate limiting (10 req/min), CORS protection, and environment-based secrets management.

02 — Deep Learning · Oct 2025

Celebrity Look-Alike Finder

End-to-end deep learning application that finds a user's top celebrity look-alike and displays a resemblance score. Processed 4K+ images from the Labeled Faces in the Wild (LFW) dataset, selecting 150+ balanced celebrity classes. Generated facial embeddings using a pretrained ResNet50 model, ranked via cosine similarity, and deployed with MTCNN face detection and a real-time Streamlit interface.

03 — Kaggle Competition · Mar 2025

Kaggle March ML Mania 2025

Predictive model for NCAA tournament outcomes, engineering features from regular season history, past tournament results, and seed data — normalizing raw team stats into per-100 possession metrics to adjust for game pace. XGBoost model with a custom Cauchy loss function predicts score differentials; spline interpolation converts predictions into win probabilities optimized for log loss. 5-fold CV over 10 repeats for stability. Ranked in the top 20% of all submissions.
